VPython Robot Mini

Proyek ini menampilkan model robot mini menggunakan VPython.

Untuk menjalankan:

  1. Pastikan Python 3 dan VPython telah terinstal.
  2. Jalankan file robot_mini.py menggunakan terminal atau VSCode.

Robot terdiri dari elemen-elemen 3D seperti box dan cylinder dengan warna dan posisi yang ditentukan.
